## ✨ About The Project

This repository contains the source code for the official website of the **Inamigos NGO Foundation**, developed to support their mission of "Uniting Minds For Change." 

Designed and engineered from the ground up during an internship, the goal of this platform is to provide a seamless digital experience for volunteers, donors, and those seeking assistance—effectively bridging the gap between available resources and marginalized communities.

### 🎯 Key Objectives Achieved:
- **Digital Presence:** Established a strong, trustworthy, and modern digital footprint for the NGO.
- **User Engagement:** Streamlined the volunteer onboarding, campaign tracking, and donation workflows.
- **Performance:** Optimized for speed, SEO, and accessibility across all devices to ensure inclusivity.
