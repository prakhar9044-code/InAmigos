## ✨ About The Project

This repository contains the source code for the official website of the **Inamigos NGO Foundation**, developed during my internship. The goal of this platform is to provide a seamless digital experience for volunteers, donors, and those seeking assistance, bridging the gap between resources and those in need.

### 🎯 Key Objectives Achieved:
- **Digital Presence:** Established a strong, modern digital footprint for the NGO.
- **User Engagement:** Streamlined the volunteer onboarding and donation workflows.
- **Performance:** Optimized for speed and accessibility across all devices.
